Pour installer l'éditeur de code (pour tous les sujets) :
Chercher vim
Dans Base, trouver vim-minimal, cliquer sur la flèche à droite de Passe, sélectionner la dernière version (8.2.4372-2 par exemple)
Dans Editors, trouver vim, cliquer sur la flèche à droite de Passe, sélectionner la dernière version (8.2.4372-2 par exemple)
Dans Editors, trouver vim-common, cliquer sur la flèche à droite de Passe, sélectionner la dernière version (8.2.4372-2 par exemple)
ATTENTION A SELECTIONNER LA MEME VERSION POUR LES 3 COMPOSANTS DE VIM !

Pour installer le compilateur C (pour les sujets en C) :
Chercher gcc
Dans Devel, trouver gcc-gcc++, cliquer sur la flèche à droite de Passe, sélectionner la dernière version (11.4.0-1 par exemple)

Terminer l'installation
